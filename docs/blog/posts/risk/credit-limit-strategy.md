# 额度策略介绍

本文主要针对信贷领域的额度做一个简单介绍，主要包含额度相关理论和策略、额度的调整等。

<!-- more -->

## 1. 额度介绍

授信额度
: 位于整个贷前策略的尾部，对未命中贷前策略的用户进行授信额度策略制定。初始额度的制定应考虑控制 **风险、市场竞争以及收益** 三方面的平衡，在保持适当市场竞争力的同时尽可能低，然后再通过跟踪监测贷中行为逐步进行提高。

额度类型
: 以信用卡为例，额度类型可以分为固定额度、临时额度、专项额度。一般临时额度是固定额度的 20%-50%，有效期为 3 个月以内；而专项额度是“隐藏额度”，是银行专门提供大额消费现金的额度，不占用前面两项额度。

额度管理
: 可分为 **状态管理、调控管理以及使用管理**：
: - 状态管理包括额度申请、额度生效、额度暂停、额度终止、额度恢复、额度撤销、额度作废；
: - 调控管理包括额度修改、额度冻结、额度解冻；
: - 使用管理包括额度检查、额度圈存、额度解圈存、额度占用、额度释放、额度共用。
: 以信用卡为例，额度冻结是指可用额度冻结起来，额度冻结以后信用卡无法正常使用。而额度圈存是将额度圈存到电子现金业务，占用额度，但是不消费的话就不计入账单，无需还款。
