# 提前结清、久期和利润测算

> 原文地址：https://mp.weixin.qq.com/s/efs8zqt-bI1mt1xYjI03sw

不同提前结清情况下，贷款久期如何变化以及利润测算时应如何考虑提前结清损失部分的影响，本文结合工作实践进行了思考和整理。

<!-- more -->

## 1. 提前损失和久期计算

假设一款产品，额度1万元，分12期，等额本金，年化利率24%(APR下为13%)，提前结清无手续费和违约金。还款计划表如下：

![](https://mingminyu.github.io/webassets/images/20251111/01.png)

上图中，最右边一列是每一期的提结损失，计算公式为 `(应收利息-已收利息)/本金`，如果用户12期都正常还款，则提结损失为 0，这种情况下该笔贷款的久期为 6.5。

如果用户第1期时发生提前结清，仅支付第一期的利息，则真实的还款计划表如下：

![](https://mingminyu.github.io/webassets/images/20251111/02.png)

虽然名义利率为 13%，但是由于提前结清损失为 11%，提前结清损失了利息收入，所以实际只收到了 (13%-11%)=2% 的利息，这种情况下的久期为 1。

同理，如果结清发生在 1-12 期的每一期，则都会有一个相对应的提前结清的损失和实际的久期：

| |  提前结清损失   |  实际久期  |
| :----: | :----: | :----: |
| 1期 | 11% | 1 |
| 2期 | 9.17% | 1.92 |
| 3期 | 7.50% | 2.75 |
| 4期 | 6.00% | 3.50 |
| 5期 | 4.67% | 4.17 |
| 6期 | 3.50% | 4.75 |
| 7期 | 2.50% | 5.25 |
| 8期 | 1.67% | 5.67 |
| 9期 | 1.00% | 6.00 |
| 10期 | 0.50% | 6.25 |
| 11期 | 0.17% | 6.42 |
| 12期 | 0.00% | 6.50 |

对于一整包资产来说，会有不同比例的用户在不同mob发生提前结清，假设提结人群的金额分布如下：

| 期数 (mob) | 提前结清金额占比 (累计) | 每期结清占比 (当期) | 久期 (等额本金) |
| ---------- | ----------------------- | ------------------- | --------------- |
| mob1       | 5.00%                   | 5.00%               | 1.00            |
| mob2       | 10.00%                  | 5.00%               | 1.92            |
| mob3       | 14.00%                  | 4.00%               | 2.75            |
| mob4       | 17.00%                  | 3.00%               | 3.50            |
| mob5       | 19.00%                  | 2.00%               | 4.17            |
| mob6       | 20.00%                  | 1.00%               | 4.75            |
| mob7       | 21.00%                  | 1.00%               | 5.25            |
| mob8       | 22.00%                  | 1.00%               | 5.67            |
| mob9       | 23.00%                  | 1.00%               | 6.00            |
| mob10      | 24.00%                  | 1.00%               | 6.25            |
| mob11      | 25.00%                  | 1.00%               | 6.42            |
| mob12      | 26.00%                  | 1.00%               | 6.50            |
|            |                         |                     | **5.66**        |

约有26%的本金会在12期以内发生提结，用每一期结清金额和结清损失进行加权求和，就可以得到提结结清的损失为 1.68%。这里由于提前结清只损失利率，所以这个提结损失仅代表提前结清对利息的影响。

久期的计算会多一步，除了对结清部分用结清金额对每一期的久期进行加权之外，还要再加上未结清部分的久期 6.5，把提前结清和未提前结清按照金额占比对久期进行加权求和，计算结果为 5.66。

## 2. 单体和年化利润率计算

分别对提结人群、未提结人群、整包资产计算单体和年化利润率情况。

整体

| 指标项            | 单体   | 年化       |
| ----------------- | ------ | ---------- |
| **收费**          | 13.00% | 27.55% |
| **期限（月）**    | 12.00  | 12.00      |
| **久期（月）**    | 5.66   | 5.66       |
| **Vintage逾期率** | 4.00%  | 8.48%  |
| **提前还款损失**  | 1.68%  | 3.55%  |
| **资金成本**      | 2.7%   | 5.7%   |
| **获客成本**      | 3.0%   | 6.4%   |
| **利润率**        | 1.63%  | 3.44%  |

提前结清客群（26%）

| 指标项            | 单体   | 年化       |
| ----------------- | ------ | ---------- |
| **收费**          | 13.00% | 47.58% |
| **期限（月）**    | 12.00  | 12.00      |
| **久期（月）**    | 3.28   | 3.28      |
| **Vintage逾期率** | 0.00%  | 0.00%  |
| **提前还款损失**  | 6.44%  | 23.58%  |
| **资金成本**      | 2.7%   | 9.9%   |
| **获客成本**      | 3.0%   | 10.98%   |
| **利润率**        | 0.86%  | 3.14%  |


未结清客群（26%）

| 指标项            | 单体   | 年化       |
| ----------------- | ------ | ---------- |
| **收费**          | 13.00% | 24% |
| **期限（月）**    | 12.00  | 12.00      |
| **久期（月）**    | 6.50   | 6.50      |
| **Vintage逾期率** | 5.41%  | 9.98%  |
| **提前还款损失**  | 0.00%  | 0.00%  |
| **资金成本**      | 2.7%   | 5.0%   |
| **获客成本**      | 3.0%   | 5.54%   |
| **利润率**        | 1.89%  | 3.50%  |


单体口径下，三种计算方式下的利润率可以准确对上，`1.63%=26%*0.86%+74%*1.89%`；年化口径下，会有很小的差异，`26%*3.14%+74%*3.5%=3.4%`，实际整体上算出来是 3.44%。

值得注意的是，整体APR利率是 13%，年化利率是 27.55%，不是 24%。这是因为24%和13%是在久期为6.5的情况下得到的。如果有提前结清，则久期变短至5.6，资金的周转效率变快，也就是说有些钱可以提前收回再拿去放贷，所以实际的年化利率会更高一些。

## 3. 一些思考

1. 名义久期、提前结清速率、实际久期关系

    上文中名义久期6.5，提前结清26%，实际久期为 5.6。笔者有了解到，有些公司是用 `名义久期*(1-提前结清)` 作为实际的久期，在上文的例子中不能很好对上，主要是因为久期的计算除了和最终结清比例，还和每一期的具体结清分布有关。所以这种方式只能是比较粗略的一种估法。

2. 真实提前结清损失

    上文中提前结清损失只考虑了利息的损失，实际提前结清会收取一定比例的违约金，在实际业务中这部分也需要考虑进去。

3. 对策略测算的影响

    对策略命中的一小部分人进行单体测算时，需要看这部分人结清金额在mob上的分布，再结合每一期结清的结清损失、久期来计算得到这部分人整体的提前结清损失、久期。后两者都是相对固定的，所以主要关键还是在于结清金额在 mob 上的分布情况。

4. 别的测算方法

    由于提前结清损失这一项涉及利息、违约金的收法，实际情况可能比较复杂。为了简化计算，笔者了解到有些公司会直接把vintage 损失，按照实际久期折算之后，当成年化风险，不再单独考虑提结损失。实际算出来的值会有些偏高
